# software_development_skills

English | [中文](README.md)

A set of software development workflow skill documents for AI, covering the complete lifecycle from requirements analysis to delivery.

## Project Structure

- `software-workflow` — Full workflow entry point, orchestrates all stages
- `requirements-analysis` — Requirements analysis
- `software-design` — Software design
- `dev-planning` — Development planning and preparation
- `coding` — Coding
- `code-review` — Code review
- `task-breakdown` — Task breakdown
- `testing` — Testing
- `deployment` — Deployment
- `rules` — Shared rules (coding standards, Git workflow, review, etc.)
- `writing-skills` — Skill authoring guide

## Installation & Usage

1. Create a directory: create an empty directory as the project workspace and open it in your AI tool
2. Install the skill: `npx skills add yinhunfeixue/software_development_skills --skill '*'`
3. Get started:
   - Type `load skill` in chat
   - Then describe your requirement, e.g. `I want to create a news website`
