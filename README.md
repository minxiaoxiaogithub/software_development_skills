# software_development_skills

[English](README.en.md) | 中文

一套给 AI 使用的软件开发全流程 skill 文档，覆盖从需求分析到交付的完整链路。

## 项目结构

- `software-workflow` — 全流程入口，串联所有阶段
- `requirements-analysis` — 需求分析
- `software-design` — 软件设计
- `dev-planning` — 开发规划和准备
- `coding` — 编码
- `code-review` — 代码 Review
- `task-breakdown` — 任务拆解
- `testing` — 测试
- `deployment` — 部署
- `rules` — 通用规范（代码规范、Git 工作流、Review 机制等）
- `writing-skills` — Skill 编写指南

## 安装与使用

1. 创建目录：新建一个空目录作为项目 workspace，在 AI 工具中打开
2. 安装 skill：`npx skills add yinhunfeixue/software_development_skills --skill '*'`
3. 开始：
   - 在 chat 中输入 `加载 skill`
   - 然后输入你的需求，例如 `我要创建一个新闻网站`
